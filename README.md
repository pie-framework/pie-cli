# pie-cli 


## Install
```bash
npm install -g pie-cli 
```

## Usage 

```
pie --help|-h
```

### Developing

If you want to play with the cli while developing you can watch the src, then link the repo so you can invoke the cli

```
npm link
babel --watch src -d lib
# in some other dir 
pie-lib --help
```

#### Tests

##### Unit
```
npm test
```

##### Integration 

```
npm run it 
```

#### Build

```
npm run build
```
